# High Performance Computing

This research group aims to develop new solutions for the high performance computing area. in particular, solutions for the Concept Drift and Reinforcement Learning areas. We seek to develop parallel and distributed solutions from their conception, exploring the wide possibilities of available hardware.

## Featured Projects

**Easy Stream Framework** In this project we developed a framework for data streams. This framework presents a set of interfaces that make the prototyping and performance evaluation easier. Since our goal is fast coding and runtime efficiency, we use the Julia language in this project. 

**High Performance Algorithms for Data Streams** In this project we seek to develop new parallel or distributed methods for data streams that incorporate concept drift techniques. We also use Julia language as our coding language base.  

**Reinforced Learning and Neuroevolution** In this project we developed new representation solutions for evolutionary networks with the objective of gains in performance or memory efficiency. Here we explore mathematical representations in different types of hardware and different types of data.

## Team

We are a group with complementary knowledge that encourages the exchange of ideas from the beginning regardless of the training of each individual. We care about the formation of our members from their entry into the group until their departure. We designed a learning cycle in order to accelerate the student's maturity for academic and practical projects. We presented early development tools, versioning, good programming practices and lectures.
